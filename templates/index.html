How this Webapp works:

This webapp will allow the user to input a .txt file containing the potential drug compounds, then uses a pre-trained machine learning model to predict the pIC50 inhibition value of the compounds on the protein. The results will be stored in a .csv file and downloadable.

What is pIC50 value:

Half maximal inhibitory concentration (IC50) is a measure of the potency of a subtance inhibiting a specific biological or biochemical function. It measures the concentration of the substance needed to inhibit the protein by 50%. pIC50 value is the negative log of IC50 and is typically a better indicator than IC50. A higher pIC50 value indicates a stronger inhibition interaction.
